## 🎨 Caractéristiques techniques

✅ HTML5 sémantique pur (sans CSS ni JavaScript)  
✅ Balises structurelles : `<header>`, `<main>`, `<aside>`, `<footer>`, `<section>`, `<article>`  
✅ Métadonnées complètes (charset UTF-8, viewport responsive)  
✅ Listes sémantiques : listes à puces `<ul>`, listes ordonnées `<ol>`, listes de définitions `<dl>`  
✅ Citations : utilisation de `<blockquote>` pour les pensées inspirantes  
✅ Liens fonctionnels : `mailto:` (email), `tel:` (téléphone), réseaux sociaux  
✅ Attributs de sécurité : `rel="noopener"` et `rel="noopener noreferrer"` sur les liens externes  
✅ Images responsive : attributs `alt`, `width`, `height`  
✅ Dates sémantiques : balises `<time>` avec `datetime`
